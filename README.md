# Nivis App

A mobile app for teaching Sorani Kurdish to English speakers.

## Goal
Help users:
- Learn Sorani script (Arabic-based)
- Build vocabulary
- Practice speaking

## Tech
- Flutter (frontend)
- FastAPI (backend later)
- AI APIs (later)

## First Milestone
Build an Alphabet Trainer:
- Show Sorani letters
- Tap letter → play sound
